💻 Integration Code Example

<!-- React Component Usage --> import React from 'react'; import { ChatComponent } from '@inprove/chat-component'; function App() { return ( <div> <h1>My Website</h1> <ChatComponent apiUrl="https://api.inproveapp.com" apiKey="your-api-key-here" userId="user123" theme="light" height="400px" width="100%" onMessage={(message) => console.log('New message:', message)} onError={(error) => console.error('Chat error:', error)} /> </div> ); }

🔒 Security Features

  • Domain Validation: Only authorized domains can use the component
  • API Key Authentication: All requests require valid API keys
  • CORS Protection: Backend configured to only accept requests from authorized domains
  • Input Sanitization: All user inputs are sanitized to prevent XSS
